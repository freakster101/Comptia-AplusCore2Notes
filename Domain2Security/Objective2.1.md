# 2.1 Security Measures and Their Purposes

# Physical Security

## Bollards
Physical barriers used to stop unauthorized vehicle access.  
Example: Concrete posts outside a government building.

## Access Control Vestibule
Secured entry area allowing controlled access one person at a time.  
Example: Mantrap entrance at a data center.

## Badge Reader
Device that scans employee badges for authentication and entry.  
Example: Staff tapping ID badges to enter office floors.

## Video Surveillance
Cameras used to monitor and record activities for security purposes.  
Example: CCTV cameras monitoring parking lots.

## Alarm Systems
Systems that alert security personnel during unauthorized access or emergencies.  
Example: Burglar alarm triggered after office hours.

## Motion Sensors
Devices that detect movement in restricted or monitored areas.  
Example: Motion detector activating security lights.

## Door Locks
Physical locking mechanisms used to secure rooms and buildings.  
Example: Deadbolt lock on a server room door.

## Equipment Locks
Locks designed to physically secure devices against theft.  
Example: Kensington lock attached to a laptop.

## Security Guards
Personnel responsible for monitoring and protecting facilities and assets.  
Example: Guard checking visitor IDs at reception.

## Fences
Physical barriers used to restrict unauthorized access to property.  
Example: Metal fence surrounding a warehouse.

---

# Physical Access Security

## Key Fobs
Electronic devices used for secure access authentication.  
Example: RFID key fob unlocking office entrance.

## Smart Cards
Cards with embedded chips used for secure authentication.  
Example: Employee smart card used for building access.

## Mobile Digital Key
Smartphone-based credential used to unlock secure areas.  
Example: Mobile app unlocking hotel room doors.

## Keys
Traditional physical devices used to unlock doors or cabinets.  
Example: Metal office door key.

---

# Biometrics

## Retina Scanner
Authentication system using retina blood vessel patterns for identity verification.  
Example: Retina scanner at military facilities.

## Fingerprint Scanner
Authentication system using fingerprint patterns for access control.  
Example: Smartphone fingerprint unlock.

## Palm Print Scanner
Authentication system using palm characteristics for identification.  
Example: Palm scanner at secure research labs.

## Facial Recognition Technology (FRT)
Uses facial features to identify and authenticate users.  
Example: Face unlock on modern smartphones.

## Voice Recognition Technology
Uses voice patterns for identity verification and authentication.  
Example: Voice authentication for banking systems.

---

# Additional Physical Security

## Lighting
Improves visibility and deters unauthorized activity around facilities.  
Example: Floodlights around building entrances.

## Magnetometers
Devices used to detect concealed metal objects for screening purposes.  
Example: Airport metal detectors.

---

# Logical Security

## Principle of Least Privilege
Users receive only the minimum permissions necessary for their job role.  
Example: Employee can view files but cannot delete them.

## Zero Trust Model
Security model that assumes no user or device is trusted automatically.  
Example: MFA required even inside the corporate network.

## Access Control Lists (ACLs)
Rules that define permitted or denied access to resources.  
Example: Firewall ACL blocking unauthorized IP addresses.

## Multifactor Authentication (MFA)
Security method requiring multiple forms of authentication.  
Example: Password plus fingerprint login.

---

# MFA Methods

## Email
Authentication code delivered through email verification.  
Example: Login code sent to Gmail account.

## Hardware Token
Physical device generating authentication codes for secure login.  
Example: RSA security token.

## Authenticator Application
Mobile application generating temporary verification codes.  
Example: Google Authenticator app.

## Short Message Service (SMS)
One-time verification code sent through text messaging.  
Example: Bank sending OTP via SMS.

## Voice Call
Authentication code delivered through automated phone calls.  
Example: Security verification by voice call.

## Time-based One-time Password (TOTP)
Temporary code generated based on synchronized time intervals.  
Example: 30-second rotating code in Authenticator app.

## One-time Password/Passcode (OTP)
Single-use authentication code for secure access.  
Example: One-time banking verification code.

---

# Identity and Access Technologies

## Security Assertion Markup Language (SAML)
Authentication standard enabling secure identity sharing between services.  
Example: Logging into Salesforce using Microsoft account credentials.

## Single Sign-on (SSO)
Allows users to access multiple systems using one login session.  
Example: One login for Gmail, Drive, and Calendar.

## Just-in-time Access
Provides temporary elevated permissions only when needed.  
Example: Admin granted privileges for one hour during maintenance.

## Privileged Access Management (PAM)
Controls and monitors administrator-level accounts and privileges.  
Example: Storing admin credentials inside a secure vault.

## Mobile Device Management (MDM)
Centralized platform used to manage and secure mobile devices.  
Example: IT remotely wiping a lost company phone.

## Data Loss Prevention (DLP)
Prevents unauthorized transfer or leakage of sensitive data.  
Example: Blocking employees from emailing confidential documents.

## Identity Access Management (IAM)
Framework used to manage user identities and access permissions.  
Example: Creating employee accounts with role-based access.

## Directory Services
Centralized system used to manage users, groups, and network resources.  
Example: Microsoft Active Directory.
