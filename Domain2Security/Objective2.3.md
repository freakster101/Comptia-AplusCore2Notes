# 2.3 Wireless Security Protocols and Authentication Methods

# Protocols and Encryption

## Wi-Fi Protected Access 2 (WPA2)
Wireless security protocol providing strong encryption using AES.  
Example: Securing home Wi-Fi network with WPA2-Personal.

## WPA3
Modern wireless security protocol offering stronger encryption and improved authentication security.  
Example: New router configured with WPA3 protection.

## Temporal Key Integrity Protocol (TKIP)
Older wireless encryption protocol created as a temporary replacement for WEP.  
Example: Legacy Wi-Fi devices using WPA with TKIP.

## Advanced Encryption Standard (AES)
Strong modern encryption standard commonly used with WPA2 and WPA3.  
Example: Enterprise Wi-Fi secured using WPA2-AES.

# Authentication Methods

## Remote Authentication Dial-in User Service (RADIUS)
Centralized authentication service commonly used for network and Wi-Fi access control.  
Example: Employees authenticating to enterprise Wi-Fi through RADIUS server.

## Terminal Access Controller Access-control System (TACACS+)
Authentication protocol primarily used for administrative access to network devices.  
Example: Network admins logging into Cisco routers using TACACS+.

## Kerberos
Ticket-based authentication protocol used for secure identity verification in Windows domains.  
Example: Active Directory users authenticated through Kerberos.

## Multifactor Authentication (MFA)
Security method requiring multiple authentication factors for access.  
Example: Password plus mobile authenticator app verification.
