# 2.8 Given a scenario, apply common methods for securing mobile devices

# Hardening Techniques

## Device Encryption

Protects stored mobile data by encrypting it against unauthorized access.
Example: Encrypting an Android phone using built-in encryption.

# Screen Locks

## Facial Recognition

Uses facial scanning for device authentication.
Example: Unlocking an iPhone using Face ID.

## PIN Codes

Uses numeric passwords to secure device access.
Example: Unlocking a phone with a 6-digit PIN.

## Fingerprint

Uses biometric fingerprint scanning for authentication.
Example: Unlocking a Samsung phone with fingerprint recognition.

## Pattern

Uses a drawn pattern on the screen for authentication.
Example: Android device unlocked using a pattern grid.

## Swipe

Basic screen gesture lock with little or no security protection.
Example: Swiping up to access the phone lock screen.

## Configuration Profiles

Preconfigured security and management settings applied to mobile devices.
Example: Company Wi-Fi and email settings automatically installed on employee phones.

# Patch Management

## OS Updates

Installs operating system security fixes and feature updates.
Example: Updating iOS to patch vulnerabilities.

## Application Updates

Keeps installed apps secure and stable with newer versions.
Example: Updating Microsoft Teams from the App Store.

# Endpoint Security Software

## Antivirus

Detects and removes viruses from mobile devices.
Example: Installing Avast Mobile Security.

## Anti-malware

Protects devices against spyware, ransomware, and malicious apps.
Example: Malwarebytes scanning Android applications.

## Content Filtering

Blocks harmful or unauthorized websites and content.
Example: Restricting access to adult websites on company devices.

# Locator Applications

Helps locate lost or stolen mobile devices using GPS tracking.
Example: Using Find My iPhone to locate a missing phone.

# Remote Wipes

Allows administrators or users to erase data remotely from lost devices.
Example: Erasing company data from a stolen smartphone.

# Remote Backup Applications

Automatically backs up mobile data to cloud or remote storage.
Example: Google Drive backing up Android photos and contacts.

# Failed Log-in Attempts Restrictions

Locks or wipes devices after repeated failed authentication attempts.
Example: Phone locking after 10 incorrect PIN attempts.

# Policies and Procedures

## MDM (Mobile Device Management)

Centralized system used to manage and secure mobile devices remotely.
Example: Company IT enforcing security policies through Microsoft Intune.

## BYOD vs. Corporate-Owned Devices

Defines whether employees use personal devices or company-issued devices for work.
Example: Employee using personal iPhone under BYOD policy.

## Profile Security Requirements

Security rules applied to user profiles and mobile device configurations.
Example: Requiring encryption and screen locks on all employee devices.
