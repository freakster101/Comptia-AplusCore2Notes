# 2.4 Malware Types and Detection/Removal/Prevention Methods

# Malware

## Trojan
Malware disguised as legitimate software to trick users into installing it.  
Example: Fake antivirus software containing malicious code.

## Rootkit
Malware designed to hide itself and provide unauthorized privileged access to a system.  
Example: Rootkit hiding malicious processes from Task Manager.

## Virus
Malicious code that attaches itself to files and spreads when executed.  
Example: Infected executable spreading through USB drives.

## Spyware
Malware that secretly monitors user activity and collects information.  
Example: Spyware recording browsing history and passwords.

## Ransomware
Malware that encrypts files and demands payment for decryption.  
Example: User files locked with a cryptocurrency ransom demand.

## Keylogger
Malware that records keyboard input to steal sensitive information.  
Example: Capturing online banking passwords.

## Boot Sector Virus
Virus that infects the boot sector or master boot record of a drive.  
Example: Malware preventing Windows from booting properly.

## Cryptominer
Malware that secretly uses system resources to mine cryptocurrency.  
Example: Infected PC showing unusually high CPU usage.

## Stalkerware
Malware used to secretly monitor another person’s device activity.  
Example: Hidden app tracking calls and GPS location.

## Fileless Malware
Malware operating primarily in memory without installing files on disk.  
Example: PowerShell-based attack running entirely in RAM.

---

# Adware

## Potentially Unwanted Program (PUP)
Unwanted software that may display ads or alter system behavior.  
Example: Browser toolbar installed with free software.

---

# Tools and Methods

## Recovery Console
Boot environment used to troubleshoot and repair damaged operating systems.  
Example: Accessing Windows Recovery Environment to repair boot issues.

## Endpoint Detection and Response (EDR)
Security solution that monitors endpoints for suspicious activity and threats.  
Example: Detecting ransomware behavior on employee laptops.

## Managed Detection and Response (MDR)
Third-party managed security service providing threat monitoring and response.  
Example: Security provider monitoring company systems 24/7.

## Extended Detection and Response (XDR)
Security platform integrating threat detection across multiple environments and devices.  
Example: Correlating email, server, and endpoint attack data together.

## Antivirus
Software designed to detect, quarantine, and remove known viruses and malware.  
Example: Microsoft Defender Antivirus scanning downloaded files.

## Anti-malware
Software focused on detecting and removing various types of malicious software.  
Example: Malwarebytes removing spyware infections.

## Email Security Gateway
Filters malicious emails, spam, and phishing attempts before reaching users.  
Example: Blocking phishing attachments at mail server level.

## Software Firewalls
Applications controlling inbound and outbound network traffic on devices.  
Example: Windows Defender Firewall blocking unauthorized connections.

## User Education Regarding Common Threats
Security awareness training teaching users how to recognize and avoid threats.  
Example: Employees learning to identify suspicious links and attachments.

### Antiphishing Training
Specialized training focused on recognizing phishing and social engineering attacks.  
Example: Simulated phishing email campaigns for employees.

## OS Reinstallation
Completely reinstalling the operating system to remove severe infections.  
Example: Reinstalling Windows after failed malware remediation.
