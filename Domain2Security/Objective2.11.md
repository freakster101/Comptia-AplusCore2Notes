# 2.11 Given a scenario, configure relevant security settings in a browser

# Browser Download/Installation

## Trusted Sources

Browsers should only be downloaded from official or verified websites.
Example: Downloading Chrome from Google’s official website.

## Hashing

Uses cryptographic hash values to verify file integrity and authenticity.
Example: Comparing SHA256 hash after downloading Firefox.

## Untrusted Sources

Downloading browsers from unknown websites increases malware risks.
Example: Fake Chrome installer from a suspicious website.

# Browser Patching

Keeping browsers updated to fix vulnerabilities and improve security.
Example: Installing the latest Microsoft Edge security update.

# Extensions and Plug-ins

## Trusted Sources

Extensions should only be installed from official browser stores or trusted vendors.
Example: Installing uBlock Origin from Chrome Web Store.

## Untrusted Sources

Unknown extensions may contain spyware or malicious code.
Example: Fake coupon extension stealing browsing data.

# Password Managers

Securely stores and autofills passwords in encrypted vaults.
Example: Using Bitwarden to manage login credentials.

# Secure Connections/Sites – Valid Certificates

Ensures websites use trusted SSL/TLS certificates for encrypted communication.
Example: HTTPS lock icon appearing in the browser address bar.

# Settings

## Pop-up Blocker

Prevents unwanted pop-up windows from appearing.
Example: Blocking fake virus alert pop-ups.

## Clearing Browsing Data

Removes stored browsing history, cookies, and saved site data.
Example: Clearing browsing history on a public computer.

## Clearing Cache

Deletes temporary website files stored locally by the browser.
Example: Clearing cache to fix website loading problems.

## Private-Browsing Mode

Prevents browser history and cookies from being saved locally.
Example: Using Incognito Mode on a shared PC.

## Sign-in/Browser Data Synchronization

Synchronizes bookmarks, passwords, and settings across devices.
Example: Chrome syncing bookmarks between laptop and phone.

## Ad Blockers

Blocks advertisements and potentially malicious ad content.
Example: Using Adblock Plus to reduce unwanted ads.

## Proxy

Routes browser traffic through an intermediary server for filtering or privacy.
Example: Company browser traffic passing through a proxy server.

## Secure DNS

Uses encrypted DNS requests for safer domain name resolution.
Example: Enabling DNS over HTTPS in Firefox.

# Browser Feature Management

## Enable/Disable Plug-ins

Controls whether browser plug-ins are active or inactive.
Example: Disabling outdated Java plug-ins.

## Enable/Disable Extensions

Turns browser extensions on or off for security or troubleshooting.
Example: Disabling a malfunctioning extension causing crashes.

## Enable/Disable Features

Manages built-in browser functions and capabilities.
Example: Disabling browser notifications for unknown sites.
