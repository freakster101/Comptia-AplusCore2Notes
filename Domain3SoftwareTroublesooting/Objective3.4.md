# 3.4 PC Security Troubleshooting Quick Notes

## Common Symptoms

### Unable to Access the Network

**Note:** Check network settings, firewall rules, malware infection, and adapter configuration.
**Example:** Malware changes DNS settings, preventing access to websites.

---

### Desktop Alerts

**Note:** Unexpected desktop alerts may indicate adware, scareware, or malware infection.
**Example:** Pop-ups constantly appear claiming the computer is at risk.

---

### False Alerts Regarding Antivirus Protection

**Note:** Fake security software often displays false infection warnings to trick users.
**Example:** A popup claims "500 viruses detected" and demands payment.

---

### Altered System or Personal Files

**Note:** Malware may modify, encrypt, or corrupt files without user permission.
**Example:** Documents become encrypted after a ransomware attack.

---

### Missing/Renamed Files

**Note:** Malware can hide, delete, or rename files to disrupt system operation.
**Example:** User documents suddenly have unfamiliar file extensions.

---

### Inability to Access Files

**Note:** Check file permissions, encryption, corruption, or malware activity.
**Example:** A user receives "Access Denied" when opening important files.

---

### Unwanted Notifications Within the OS

**Note:** Excessive notifications may result from adware or malicious applications.
**Example:** The operating system constantly displays advertisements and promotions.

---

### OS Update Failures

**Note:** Malware, corruption, or security settings may prevent updates from installing.
**Example:** Windows Update repeatedly fails because malicious software disabled update services.

---

# Browser-Related Symptoms

### Random/Frequent Pop-Ups

**Note:** Frequent pop-ups are commonly caused by adware or malicious browser extensions.
**Example:** Advertisements appear even when no websites are open.

---

### Certificate Warnings

**Note:** Certificate warnings may indicate an invalid website certificate or a possible security threat.
**Example:** A browser warns that a banking website's certificate is not trusted.

---

### Redirection

**Note:** Unexpected redirection often results from malware, browser hijackers, or malicious extensions.
**Example:** Searching with Google automatically redirects to a suspicious website.

---

### Degraded Browser Performance

**Note:** Slow browser performance may be caused by malware, excessive extensions, or high resource usage.
**Example:** Web pages take several minutes to load because of a malicious toolbar extension.
