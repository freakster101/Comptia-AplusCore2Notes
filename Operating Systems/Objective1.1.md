# 1.1 Explain common operating system (OS) types and their purposes.
## Workstation systems (OSs)
A workstation OS is designed for everyday personal or business computer use.
````bash
Example: Windows 11 on an office desktop.
````
### Windows
Microsoft's operating system used for gaming, business, and general computing.
````bash
Example: Windows 11 on HP laptop
````
 
### Linux
An open-source operating system popular for server. cybersecurity, and developers.
````bash
Example: Ubuntu Linux running on web server
````

### macOS
Apple's operating system designed for Mac computers.

````bash
Example: macOS Sonoma on a MacBook Pro.
````

### Chrome OS
A lightweight Google operating system focused mainly on web applications.
````bash
Example: Chromebook used in Schools.
````

## Mobile OSs
Operating systems designed for smartphones and tablets.
````bash
Example: Android and iOS on mobile devices.
````

### iPadOS
Apple's tablet operating system optimized for multitasking and Apple Pencil support.
````bash
Example iPADOS on an iPAD Air.
````

### iOS
Apple's mobil operating system made for iPhones.
````bash
Example: iOS 18 on an iPhone.
````

### Android
Google's mobile oprating system used by many smartphone manufacturers.
````bash
Example: Samsung Galaxy phone running Android.
````

## Various filesystem types
### New Technology File System (NTFS)
Windows default file system that suppoorts permissions, encryption, and large files.
````bash
Example: Windows 11 internal SSD formatted with NtFS.
````

### Resilient File System (ReFS)
Microsoft file system designed for data protection and large enterprise storage.
````bash
Example: Windows Server backup storage using ReFS.
````

### File Allocation Table 32 (FAT32)
Older file system with broad compatibility but a 4gB file size limit.
````bash
Example: Small USB flash drive formatted as FAT32.
````

### Fourth extended filesystem (ext4)
Common Linux file system known for stability and journaling.
```bash
Example: Ubuntu installed on an ext4 partition.
````

### Extended filesystem (XFS)
High-performance Linux file system designed for large files and fast throughput.
````bash
Example: Media server storing huge video files on XFS.
````

### Apple File System (APFS)
Apple's modern file system optimized for SSDs and encryption.
````bash
Example: MacBook SSD formatted with APFS
````

### Extensible File Allocation Table (exFAT)
Cross-platform file system designed for large removeable storage devices.
````bash
Example: External SSD shared between Windows and macOS.
````

## Vendor life-cycle limitations
### End-of-life (EOL)
When a vendor stops supporting or updating a product or operating system.
````bash
Example: Windows 7 reaching end-of-life support.
````

### Update limitations
Older hardware or software may not support the latest OS updates.
````bash
Example: An old laptop unable to install Windows 11 because it lacks TPM 2.0.
````

### Compatibility concerns between operating system
Different operating systems may have problems sharing apps, files, or hardware support.
````bash
AirDrop works between Apple devices but not directly with Windows PCs.
```
