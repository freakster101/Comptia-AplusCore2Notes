# 1.11 Given a scenario, install and configure cloud-based productivity tools

# Email Systems
Cloud-based email platforms used for communication and organization.  
Example: Configuring Microsoft Outlook with Microsoft 365.

# Storage

## Sync/Folder Settings
Automatically synchronizes files between local devices and cloud storage.  
Example: OneDrive syncing Desktop and Documents folders.

# Collaboration Tools

## Spreadsheets
Cloud applications used for creating and editing spreadsheet data collaboratively.  
Example: Multiple users editing Google Sheets together.

## Videoconferencing
Cloud-based tools used for online meetings and video communication.  
Example: Joining a Zoom or Microsoft Teams meeting.

## Presentation Tools
Applications used to create and share presentations online.  
Example: Creating slides using Google Slides.

## Word Processing Tools
Cloud applications used to create and edit text documents collaboratively.  
Example: Editing documents in Microsoft Word Online.

## Instant Messaging
Real-time communication platforms for team collaboration.  
Example: Sending messages through Slack or Microsoft Teams.

# Identity Synchronization
Synchronizes user accounts and credentials between local systems and cloud services.  
Example: Active Directory syncing with Microsoft Entra ID.

# Licensing Assignment
Assigns software or service licenses to users for authorized access.  
Example: Assigning a Microsoft 365 license to an employee account.
