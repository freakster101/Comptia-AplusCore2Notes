# CompTIA A+ Core 2 Notes

This repository contains my personal study notes, Linux practice, troubleshooting labs, and command references for the CompTIA A+ Core 2 certification.

The goal of this repository is not just to pass the exam, but to build real-world IT support, Linux, networking, troubleshooting, and system administration skills.

## Topics Covered

- Operating Systems
- Linux Commands
- Windows Administration
- Security Fundamentals
- Malware Removal
- Troubleshooting
- Networking Basics
- System Utilities
- File Permissions
- User Management
- Command Line Tools
- Git & GitHub Basics
- Virtual Machines
- IT Support Labs
